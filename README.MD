Inicializar node: npm init -y
Baixar dependencias:
  * typescript -D
  * @types/node -D
  * fastify
  * prisma -D
Iniciar o typescript:
  * npx tsc --init

Lib para converter o codigo .js para o .tse executar o codigo com node de maneira automatizada:
  * tsx -D

Iniciar o prisma:
  * npx prisma init
